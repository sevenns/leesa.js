<p align="center">Small framework for standard frontend development.</p>

### - Requirements

You need [nodejs](https://nodejs.org/en/) with npm (LTS recommended)

### - Setting package.json

``` json5
{
  "scripts": {
    "dev": "leesa",
    "build": "leesa build",
    "start": "leesa start",
    "icons": "leesa icons"
  }
}
```
